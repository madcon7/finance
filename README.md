# 💼 Финансовое приложение — Инструкция

Локальное веб-приложение для учета активов, денег, налогов и деклараций.  
Все данные хранятся **только на вашем компьютере**.

---

## 🚀 Как запустить

1. Откройте терминал: `Win + R` → `cmd` → Enter
2. Вставьте команду:
```
cd C:\Users\7madc\Desktop\finance\app && npm run dev
```
3. Подождите надпись `Ready`
4. Откройте в браузере: **http://localhost:3000**

При первом входе нажмите **"Регистрация"** → введите email и пароль.

---

## 📱 Разделы приложения

| Раздел | Что там |
|--------|---------|
| 🏠 Главная | Общий капитал, наличные, безналичные, имущество |
| 🏦 Активы | Квартиры, авто, акции, крипта — все имущество |
| 💸 Транзакции | Доходы, расходы, покупки и продажи активов |

| 📋 Декларации | Сданные декларации и что нужно сдать |
| 🧾 Налоги | Расчет налогов с продажи активов |
| ⚠️ Предупреждения | Все проблемы в одном месте |
| 📥 Импорт Excel | Загрузить данные из Excel файла |

---

## 💰 Что считается "реальными деньгами"

**Денежный баланс** = наличные + безналичные деньги  
**Капитал** = денежный баланс + стоимость имущества

Квартиры, машины, бизнес — это **имущество**, а не деньги.  
Деньги — это только то, что можно потратить прямо сейчас.

## 🏠 Как добавить актив

1. Раздел **"Активы"** → кнопка **"+ Добавить"**
2. Выберите категорию (кликните на иконку)
3. Введите название, страну, сумму
4. Для **"Смешанный"** источника — укажите отдельно наличными и безналично
5. Отметьте галочки: зарубежный, нужно задекларировать
6. Для **акций** и **крипты** — заполните дополнительные поля (компания, тикер, брокер)

---

## 🛒 Как купить актив через транзакцию

1. Раздел **"Транзакции"** → **"+ Добавить"**
2. Тип: **"покупка актива"**
3. Появится блок "Данные нового актива" — заполните его
4. Сохраните → актив автоматически создастся в разделе "Активы"

---

## 💰 Как продать актив

**Способ 1 — через Активы:**
1. В карточке актива нажмите **💰** (иконка продажи)
2. Укажите дату и сумму продажи
3. Актив получит статус "Продан"
4. Если есть прибыль — автоматически создастся запись в разделе "Налоги"

**Способ 2 — через Транзакции:**
1. Тип: **"продажа актива"**
2. Выберите актив из списка
3. Укажите сумму — сразу увидите прибыль/убыток и налог

---

## 🧾 Где смотреть налоги

Раздел **"Налоги"**:
- Создаются автоматически при продаже актива с прибылью
- Ставка: 10% от прибыли
- Можно отметить как "Оплачено"

> ⚠️ Расчет предварительный. Проверьте с налоговым консультантом.

---

## 📋 Где смотреть декларации

Раздел **"Декларации"**:
- Добавьте декларацию за каждый год
- Загрузите PDF файл декларации
- Укажите статус: сдана / не сдана
- Если есть зарубежный актив без декларации — появится предупреждение

## 💾 Резервная копия

Нажмите **☰** (меню) → **"Резервная копия"**  
Файл `finance_backup_ДАТА.db` сохранится в загрузки.

Или вручную скопируйте: `finance\data\finance.db`

**Восстановление:**
1. Остановите приложение (Ctrl+C)
2. Замените `finance\data\finance.db` на резервную копию
3. Запустите заново

---

## 📥 Импорт из Excel

1. Раздел **"Импорт Excel"**
2. Загрузите файл `АКТИВЫ AT DO NOT DELETE.xlsx`
3. Проверьте и отредактируйте данные
4. Нажмите **"Импортировать"**
