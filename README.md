# 💼 Финансовое приложение — Инструкция по запуску

Это локальное веб-приложение для учета активов, транзакций и деклараций.
Все данные хранятся **только на вашем компьютере**.

---

## ✅ Требования

- **Node.js** (версия 18 или выше)
- Скачать: https://nodejs.org/en (кнопка "LTS")

---

## 🚀 Первый запуск

### Шаг 1 — Откройте терминал

Нажмите `Win + R`, введите `cmd`, нажмите Enter.

### Шаг 2 — Перейдите в папку приложения

```
cd C:\Users\7madc\Desktop\finance\app
```

### Шаг 3 — Установите зависимости (только один раз)

```
npm install
```

Подождите 1-2 минуты пока загружаются пакеты.

### Шаг 4 — Запустите приложение

```
npm run dev
```

### Шаг 5 — Откройте в браузере

Перейдите по адресу: **http://localhost:3000**

---

## 🔐 Регистрация

При первом открытии нажмите **"Регистрация"**, введите email и пароль.
Пароль хранится в зашифрованном виде.

## 📁 Где хранятся данные

| Что | Где |
|-----|-----|
| База данных | `finance\data\finance.db` |
| Загруженные файлы | `finance\uploads\` |

---

## 💾 Резервная копия

**Способ 1 — Через меню приложения:**
1. Откройте меню (кнопка ☰ вверху справа)
2. Нажмите "Резервная копия"
3. Файл `finance_backup_ДАТА.db` сохранится в папку Загрузки

**Способ 2 — Вручную:**
Скопируйте файл `finance\data\finance.db` в безопасное место.

---

## 🔄 Восстановление из backup

1. Остановите приложение (Ctrl+C в терминале)
2. Замените файл `finance\data\finance.db` вашей резервной копией
3. Запустите приложение заново

---

## 📥 Импорт Excel

1. Откройте раздел "Импорт" в меню
2. Нажмите "Выбрать файл"
3. Загрузите ваш Excel файл с активами
4. Проверьте и при необходимости отредактируйте данные
5. Нажмите "Импортировать"

Приложение автоматически распознает структуру файла `АКТИВЫ AT DO NOT DELETE.xlsx`.

---

## 📊 Экспорт в Excel

В разделе "Баланс" нажмите кнопку **"📥 Excel"** — скачается файл со всеми активами и транзакциями.

## ❓ Если что-то не работает

1. Убедитесь что Node.js установлен: в терминале введите `node --version`
2. Убедитесь что вы в папке `finance\app`
3. Попробуйте удалить папку `node_modules` и запустить `npm install` снова
